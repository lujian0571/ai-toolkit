# Card Header 标题规范

## 规则说明

为提升工具页面的用户体验和界面一致性，所有工具页面的卡片（Card）组件都应添加 card-header 区域，并在其中放置合适的标题。

## 应用范围

此规则适用于所有工具页面中的卡片组件，包括但不限于：
- 主要功能区域卡片
- 百科内容区域卡片
- 其他信息展示卡片

## 实施规范

1. 每个卡片必须包含 `card-header` 区域
2. `card-header` 中应包含描述性的标题
3. 标题应使用 `<h5>` 标签，class 为 `mb-0` 以消除默认下边距
4. 卡片主体内容保留在 `card-body` 区域

## 示例代码

```html
<div class="card">
    <div class="card-header">
        <h5 class="mb-0">卡片标题</h5>
    </div>
    <div class="card-body">
        <!-- 卡片主体内容 -->
    </div>
</div>
```

## 注意事项

- 保持标题简洁明了，准确反映卡片内容
- 所有同类卡片的标题层级应保持一致
- 遵循 Bootstrap 的卡片组件规范
