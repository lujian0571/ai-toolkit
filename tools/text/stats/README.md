# 字符统计工具

## 工具说明

字符统计工具是一个用于分析文本统计信息的在线工具。该工具可以统计字符数、字数、行数、段落数，并提供字符和单词频率排行。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **基本统计**：
   - 字符数统计
   - 字数统计
   - 行数统计
   - 段落数统计

2. **频率分析**：
   - 字符频率排行（Top 20）
   - 单词频率排行（Top 20）

3. **可视化展示**：
   - 卡片式数据显示基本统计
   - 表格展示频率排行
   - 响应式设计适配各种设备

## 使用说明

### 基本使用

1. 在输入框中输入需要统计的文本
2. 点击"开始统计"按钮
3. 查看各项统计数据和频率排行

### 统计指标说明

- **字符数**：文本中所有字符的数量，包括空格和标点符号
- **字数**：文本中单词的数量（以空格分隔）
- **行数**：文本中换行符的数量加一
- **段落数**：文本中空行分隔的段落数量

## 实现要点

1. 使用原生 JavaScript 实现统计分析逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 统计算法

1. **字符统计**：直接使用字符串长度属性
2. **字数统计**：按空格分割字符串并计算数组长度
3. **行数统计**：按换行符分割字符串并计算数组长度
4. **段落统计**：按双换行符分割字符串并计算数组长度
5. **频率统计**：使用对象作为哈希表统计字符/单词出现次数

### 数据展示

1. 使用卡片组件展示基本统计数据
2. 使用表格展示频率排行
3. 提供百分比显示频率占比
4. 限制频率排行显示前20个项目

## 注意事项

1. 单词是以空格分隔的连续字符序列
2. 字符频率统计区分大小写
3. 单词频率统计不区分大小写
4. 工具完全在浏览器中运行，不依赖后端服务
