# 大小写转换工具

## 工具说明

大小写转换工具是一个用于文本大小写格式转换的在线工具。该工具支持多种转换方式，包括基本的大小写转换、编程命名规范转换等。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **基本转换**：
   - 转换为大写
   - 转换为小写
   - 首字母大写
   - 标题大小写
   - 句首字母大写
   - 大小写互换

2. **编程命名规范转换**：
   - 驼峰命名法 (camelCase)
   - 帕斯卡命名法 (PascalCase)
   - 蛇形命名法 (snake_case)
   - 烤串命名法 (kebab-case)

3. **辅助功能**：
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### 基本使用

1. 在输入框中输入需要转换的文本
2. 选择合适的转换类型
3. 点击"执行转换"按钮
4. 在输出框中查看转换结果

### 连续操作

1. 使用"交换内容"按钮将输出结果移动到输入框
2. 更改转换类型
3. 再次执行转换

## 实现要点

1. 使用原生 JavaScript 实现各种转换逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 转换类型详解

1. **转换为大写**：使用 toUpperCase() 方法将所有字母转换为大写
2. **转换为小写**：使用 toLowerCase() 方法将所有字母转换为小写
3. **首字母大写**：将字符串第一个字符转换为大写
4. **标题大小写**：将每个单词的首字母转换为大写
5. **句首字母大写**：将每个句子的第一个字母转换为大写
6. **大小写互换**：遍历每个字符，大写转小写，小写转大写
7. **驼峰命名法**：移除特殊字符，第一个单词小写，后续单词首字母大写
8. **帕斯卡命名法**：移除特殊字符，每个单词首字母大写
9. **蛇形命名法**：移除特殊字符，所有字母小写，单词间用下划线连接
10. **烤串命名法**：移除特殊字符，所有字母小写，单词间用连字符连接

## 注意事项

1. 命名法转换会移除特殊字符，仅保留字母和数字
2. 工具完全在浏览器中运行，不依赖后端服务
3. 不同转换类型适用于不同场景，请根据需要选择
