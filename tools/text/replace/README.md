# 文本替换工具

## 工具说明

文本替换工具是一个用于在文本中查找并替换指定内容的在线工具。支持普通文本替换和正则表达式替换两种模式，完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **文本替换**：
   - 普通文本替换模式
   - 正则表达式替换模式
   - 支持全局替换和单次替换

2. **正则表达式支持**：
   - 忽略大小写选项 (i)
   - 全局匹配选项 (g)
   - 多行模式选项 (m)
   - 支持捕获组引用 ($1, $2, ...)

3. **辅助功能**：
   - 匹配统计：显示匹配数量
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### 普通模式替换

1. 在输入框中输入需要处理的文本
2. 在"查找文本"框中输入要查找的内容
3. 在"替换文本"框中输入替换后的内容
4. 确保选择"普通模式"
5. 点击"执行替换"按钮

### 正则表达式模式替换

1. 在输入框中输入需要处理的文本
2. 在"查找文本"框中输入正则表达式
3. 在"替换文本"框中输入替换后的内容（可使用 $1, $2 等引用捕获组）
4. 选择"正则表达式模式"
5. 根据需要选择匹配选项（忽略大小写、全局匹配、多行模式）
6. 点击"执行替换"按钮

### 统计匹配

1. 输入文本和查找内容
2. 点击"统计匹配"按钮
3. 查看匹配数量提示

## 实现要点

1. 使用原生 JavaScript 实现文本替换逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 正则表达式处理

1. 支持常见的正则表达式语法
2. 提供常用的匹配选项
3. 支持捕获组和反向引用
4. 包含正则表达式语法错误处理

| 特性 | 普通模式 | 正则模式 |
|------|----------|----------|
| 匹配方式 | 字面量匹配 | 模式匹配 |
| 特殊字符 | 无特殊含义 | 具有特殊含义 |
| 选项支持 | 有限 | 丰富选项 |
| 性能 | 较高 | 较低 |
| 功能 | 基本替换 | 高级替换 |

## 注意事项

1. 使用正则表达式时要注意语法正确性
2. 正则表达式模式下，替换文本中可使用 $1, $2 等引用捕获组
3. 工具完全在浏览器中运行，不依赖后端服务
4. 复杂的正则表达式可能会影响性能
