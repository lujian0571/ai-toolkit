# 行排序去重工具

## 工具说明

行排序去重工具是一个用于按行处理文本的在线工具，支持对文本行进行排序和去重操作。该工具完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **排序功能**：
   - 升序/降序排列
   - 字母排序、数值排序、长度排序
   - 多种排序组合方式

2. **去重功能**：
   - 去除重复行
   - 忽略大小写去重选项
   - 移除行首行尾空白字符

3. **辅助功能**：
   - 行数统计显示
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### 基本使用

1. 在输入框中输入需要处理的文本（每行一条数据）
2. 选择排序方式（升序、降序或不排序）
3. 选择排序类型（字母、数值或长度）
4. 根据需要勾选处理选项
5. 点击"执行处理"按钮
6. 在输出框中查看处理结果

### 高级选项

1. **去除重复行**：勾选后会移除内容完全相同的行
2. **忽略大小写**：去重时忽略字母大小写差异
3. **移除行首行尾空白**：清理每行前后的空格、制表符等空白字符

## 实现要点

1. 使用原生 JavaScript 实现排序和去重逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 排序算法

1. **字母排序**：使用 JavaScript 的 localeCompare 方法进行本地化字符串比较
2. **数值排序**：将字符串转换为浮点数后进行数值比较
3. **长度排序**：按字符串长度进行比较

### 去重算法

1. 使用 Set 数据结构优化去重性能
2. 支持忽略大小写的去重选项
3. 可选择是否移除行首行尾空白字符

## 注意事项

1. 数值排序时，非数值行可能会导致意外结果
2. 忽略大小写仅影响去重，不影响排序
3. 工具完全在浏览器中运行，不依赖后端服务

## 可扩展方向

1. 支持正则表达式过滤
2. 添加自定义排序规则
3. 增加批量处理功能
4. 提供处理历史记录功能
5. 支持导出处理结果为文件
