# Cron 表达式解析器

## 工具说明

Cron 表达式解析器是一个用于解析和可视化 Cron 表达式的在线工具。该工具可以解析 Cron 表达式的各个字段，并生成未来一段时间内的执行时间。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **Cron 表达式解析**：
   - 解析五字段 Cron 表达式
   - 显示各字段的详细信息
   - 支持常见的 Cron 特殊字符

2. **执行时间计算**：
   - 根据指定时间范围计算执行时间
   - 可视化显示最近执行时间
   - 支持自定义起始和结束日期

3. **表达式验证**：
   - 验证表达式格式是否正确
   - 提供清晰的错误提示

## 使用说明

### 基本使用

1. 在输入框中输入 Cron 表达式
2. 设置起始和结束日期
3. 点击"解析表达式"按钮
4. 查看表达式解析结果和最近执行时间

### Cron 表达式格式

标准格式为：`分 时 日 月 周`

例如：
- `0 0 * * *` 表示每天午夜执行
- `0 0 1 * *` 表示每月 1 日午夜执行
- `0 0 1 1 *` 表示每年 1 月 1 日午夜执行

## 技术细节

### 支持的字段

1. **分钟**：0-59
2. **小时**：0-23
3. **日期**：1-31
4. **月份**：1-12 或 JAN-DEC
5. **星期**：0-7（0 和 7 都表示星期日）或 SUN-SAT

### 支持的特殊字符

1. **\***：匹配该字段的所有值
2. **?**：不指定值
3. **-**：表示范围
4. **,**：表示列举
5. **/**：表示步长
6. **L**：表示最后
7. **W**：表示工作日
8. **#**：表示第几个星期几
