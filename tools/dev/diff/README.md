# 文本差异比较工具

## 工具说明

文本差异比较工具是一个用于比较两个文本之间差异的在线工具。该工具可以清晰地显示两个文本之间的添加、删除和修改内容，并使用不同颜色标识各类变更。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **文本差异比较**：
   - 逐行比较两个文本之间的差异
   - 高亮显示新增、删除和未变更的行
   - 清晰的视觉标识

2. **辅助功能**：
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出

## 使用说明

### 基本使用

1. 在左侧输入框中粘贴原文本
2. 在右侧输入框中粘贴需要对比的文本
3. 点击"比较差异"按钮
4. 在下方查看差异结果

### 差异标识说明

- **绿色背景 + 左侧绿线**：新增的行（在对比文本中存在，原文本中不存在）
- **红色背景 + 左侧红线**：删除的行（在原文本中存在，对比文本中不存在）
- **灰色背景**：未发生变化的行

## 实现要点

1. 使用 diff 库实现文本差异比较功能
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 差异比较算法

1. **逐行比较**：将文本按行分割后进行比较
2. **最小差异计算**：使用高效的算法计算最小差异集合
3. **变更标识**：对新增、删除和未变更的行进行不同标识

### 视觉设计

1. **颜色编码**：使用不同背景色标识不同类型的变更
2. **边框标识**：使用左侧彩色边框强化变更类型识别
3. **符号前缀**：在行首添加 +/- 符号标识增删行

## 注意事项

1. 工具完全在浏览器中运行，不依赖后端服务
2. 对于大型文本，比较可能需要一些时间
3. 差异比较基于行进行，每行的变更都会单独标识

## 可扩展方向

1. 支持字符级别的差异比较
2. 添加更多差异显示选项（如忽略空白字符等）
3. 增加差异统计信息（如新增/删除行数等）
4. 提供差异导出功能
5. 支持三个及以上文本的对比
