# 代码格式化工具

## 工具说明

代码格式化工具是一个用于美化和格式化多种编程语言代码的在线工具。该工具支持常见的编程语言，如 JavaScript、HTML、CSS、JSON 等，可以自动调整代码的缩进、空格和换行，使代码更加规范和易读。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **多语言支持**：
   - JavaScript/JSX
   - HTML
   - CSS
   - JSON
   - XML
   - SQL
   - Python
   - Java
   - C++
   - C#
   - PHP
   - Ruby
   - Go
   - Rust

2. **格式化选项**：
   - 可调节缩进大小（2/4/8个空格）
   - 统一代码风格

3. **代码高亮与行号**：
   - 格式化结果带有语法高亮显示
   - 显示代码行号，便于阅读和调试
   - 支持所有受支持语言的语法高亮

4. **辅助功能**：
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### 基本使用

1. 选择需要格式化的编程语言
2. 选择合适的缩进大小
3. 在左侧输入框中粘贴需要格式化的代码
4. 点击"格式化代码"按钮
5. 在右侧查看格式化后的结果（带有语法高亮和行号）
6. 使用"复制结果"按钮将格式化后的代码复制到剪贴板

## 实现要点

1. 使用 Prettier 库实现代码格式化功能
2. 使用 Highlight.js 库实现代码高亮功能
3. 使用 highlightjs-line-numbers.js 插件实现行号显示
4. 采用 Bootstrap 5 构建响应式界面
5. 完全在客户端运行，保护用户数据隐私
6. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 支持的格式化库

1. **Prettier**：主流的代码格式化工具
2. **Highlight.js**：流行的代码语法高亮库
3. **highlightjs-line-numbers.js**：为 Highlight.js 添加行号显示的插件
4. **多解析器支持**：针对不同语言使用对应的解析器

### 格式化选项

1. **缩进大小**：支持2、4、8个空格
2. **使用空格**：统一使用空格而非制表符
3. **分号**：默认添加分号
4. **引号**：默认使用单引号

## 注意事项

1. 工具完全在浏览器中运行，不依赖后端服务
2. 对于复杂或大型代码，格式化可能需要一些时间
3. 确保输入的代码语法正确，以获得最佳格式化效果

## 可扩展方向

1. 支持更多编程语言
2. 添加更多格式化选项（如引号类型、分号等）
3. 增加批量处理功能
4. 提供格式化历史记录功能
5. 支持导出格式化结果为文件
