# 正则表达式测试器

## 工具说明

正则表达式测试器是一个用于实时测试正则表达式匹配效果的在线工具。该工具支持多种正则表达式选项，可以查看匹配结果、捕获组和高亮显示。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **正则表达式测试**：
   - 实时测试正则表达式匹配效果
   - 支持多种匹配选项
   - 显示匹配位置和内容

2. **结果展示**：
   - 匹配结果列表
   - 捕获组详情
   - 高亮显示匹配文本

3. **匹配选项**：
   - 忽略大小写 (i)
   - 全局匹配 (g)
   - 多行模式 (m)
   - 单行模式 (s)

## 使用说明

### 基本使用

1. 在"正则表达式"框中输入要测试的正则表达式
2. 在"测试文本"框中输入要匹配的文本
3. 根据需要选择匹配选项
4. 点击"执行测试"按钮查看匹配结果

### 查看结果

1. **匹配结果**：显示所有匹配项及其位置
2. **捕获组**：显示每个匹配项的捕获组内容
3. **高亮显示**：在原文本中高亮显示匹配部分

## 实现要点

1. 使用原生 JavaScript 实现正则表达式测试逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 正则表达式选项

1. **忽略大小写 (i)**：匹配时忽略字母大小写
2. **全局匹配 (g)**：查找所有匹配项，而不是仅查找第一个
3. **多行模式 (m)**：将 ^ 和 $ 匹配应用于每一行的开头和结尾
4. **单行模式 (s)**：使 . 匹配包括换行符在内的所有字符

### 匹配结果显示

1. 匹配项列表显示匹配内容和位置
2. 捕获组详情显示每个捕获组的内容
3. 高亮显示在原文本中标注匹配部分

## 注意事项

1. 确保正则表达式语法正确
2. 工具完全在浏览器中运行，不依赖后端服务
3. 复杂的正则表达式可能会影响性能
