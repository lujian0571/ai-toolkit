# Cron 表达式生成器

## 工具说明

Cron 表达式生成器是一个用于可视化生成 Cron 表达式的在线工具。该工具通过下拉选择的方式帮助用户快速生成常用的 Cron 表达式。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **可视化生成**：
   - 提供分钟、小时、日期、月份和星期的下拉选择框
   - 实时显示生成的 Cron 表达式
   - 支持常用时间选项选择

2. **表达式操作**：
   - 一键复制生成的表达式
   - 重置功能恢复默认设置

## 使用说明

### 基本使用

1. 选择所需的分钟、小时、日期、月份和星期选项
2. 生成的表达式会实时显示在顶部
3. 点击"复制表达式"按钮将表达式复制到剪贴板
4. 点击"重置"按钮恢复默认设置

### 时间选项说明

- **分钟**：0-59 或每N分钟
- **小时**：0-23（0表示午夜，12表示中午）
- **日期**：1-31
- **月份**：1-12（1月-12月）
- **星期**：0-7（0和7都表示周日，1-6表示周一至周六）

## 实现要点

1. 使用原生 JavaScript 实现表达式生成逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 表达式格式

生成的表达式遵循标准五字段格式：`分 时 日 月 周`

例如：
- `0 0 * * *` 表示每天午夜执行
- `0 0 1 * *` 表示每月 1 日午夜执行
- `0 0 1 1 *` 表示每年 1 月 1 日午夜执行

### 特殊值支持

1. **\***：匹配该字段的所有值（默认值）
2. **/**：表示步长，如 `*/5` 表示每5分钟
3. **具体数值**：指定具体的分钟、小时、日期等

## 注意事项

1. 工具完全在浏览器中运行，不依赖后端服务
2. 本工具仅支持标准的五字段 Cron 表达式
3. 生成的表达式适用于大多数支持 Cron 的系统
