# JSON格式化工具

JSON格式化工具是一个用于处理JSON数据的实用工具，支持将压缩的JSON格式化为易读形式，或将格式化的JSON压缩为单行。

## 功能特点

### JSON格式化
- 将压缩的JSON字符串转换为具有适当缩进和换行的易读格式
- 支持自定义缩进大小
- 保持JSON数据的完整性和正确性

### JSON压缩
- 将格式化的JSON字符串压缩为单行，去除所有不必要的空白字符
- 减少JSON数据的体积，便于传输和存储
- 保持JSON数据的完整性和正确性

### 辅助功能
- 一键复制输入或输出内容到剪贴板
- 清空输入或输出内容
- 交换输入和输出内容，便于连续处理

## 技术实现

### 核心功能
- 使用原生JSON.parse()和JSON.stringify()方法处理JSON数据
- 通过JSON.stringify()的第三个参数控制格式化时的缩进
- 所有处理均在浏览器端完成，确保数据安全

### 用户体验
- 实时错误检测和提示
- 直观的双栏布局设计
- 响应式界面，适配不同屏幕尺寸
- 清晰的操作反馈

## 使用说明

1. **输入JSON数据**：
   - 在左侧输入框中粘贴或输入需要处理的JSON内容

2. **格式化操作**：
   - 点击"格式化"按钮将压缩的JSON转换为易读格式

3. **压缩操作**：
   - 点击"压缩"按钮将格式化的JSON压缩为单行

4. **辅助操作**：
   - 使用"复制"按钮将内容复制到剪贴板
   - 使用"清空"按钮清除输入或输出内容
   - 使用"交换"按钮将输出结果交换到输入框，进行进一步处理

## 工具规范

### 菜单要求
- 工具应归类于"JSON工具"分类下
- 菜单项名称为"JSON格式化"
- 面包包屑导航应为：JSON工具 / JSON格式化

### 界面要求
- 使用卡片组件展示内容，每个卡片包含 card-header 区域
- 使用 Bootstrap 5 作为 UI 框架
- 使用公共样式和脚本文件
- 采用双栏布局，左右分别显示输入和输出区域

### 功能要求
1. 核心功能：
   - JSON格式化（美化）
   - JSON压缩（最小化）

2. 辅助功能：
   - 复制输入内容
   - 复制输出内容
   - 清空输入内容
   - 清空输出内容
   - 交换输入输出内容

3. 技术实现要求：
   - 使用原生JSON API处理数据
   - 所有处理在浏览器端完成，不上传任何数据到服务器
   - 提供友好的错误提示

## 注意事项

1. 输入的内容必须是有效的JSON格式，否则会提示错误
2. 所有处理均在浏览器本地完成，不会上传到任何服务器
3. 处理大型JSON可能会消耗一定时间，请耐心等待
4. 建议在处理前备份重要的JSON数据
