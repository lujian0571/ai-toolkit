JSON格式化与压缩
使用说明
功能介绍
  • JSON格式化：将压缩的JSON字符串格式化为易读的形式，带有适当的缩进和换行
  • JSON压缩：将格式化的JSON字符串压缩为单行，去除所有不必要的空白字符
使用方法
  1. 在左侧输入框中粘贴或输入需要处理的JSON内容
  2. 点击"格式化"按钮将JSON格式化为易读形式，或点击"压缩"按钮将JSON压缩为单行
  3. 处理结果将显示在右侧输出框中
  4. 可以使用"复制"按钮将内容复制到剪贴板
  5. 使用"交换"按钮可以将输出结果交换到输入框，进行进一步处理
注意事项
  • 确保输入的内容是有效的JSON格式，否则会出现错误提示
  • 处理大型JSON可能会消耗一定时间，请耐心等待
  • 所有处理均在浏览器本地完成，不会上传到任何服务器
