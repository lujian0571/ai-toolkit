# RSA 密钥生成工具

## 工具说明

RSA 密钥生成工具是一个用于在浏览器中生成 RSA 公钥/私钥对的在线工具。该工具使用 Forge 库在客户端生成密钥对，完全在浏览器中运行，不会将密钥信息发送到任何服务器，确保用户密钥的安全性。

## 核心功能

1. **RSA 密钥对生成**：
   - 支持 1024/2048/4096 位密钥长度
   - 生成 PEM 格式的公钥和私钥
   - 完全在浏览器中运行，保障密钥安全

2. **密钥管理**：
   - 一键复制公钥或私钥
   - 清晰的密钥显示区域
   - 安全提醒和使用指导

## 使用说明

### 生成密钥对

1. 选择所需的密钥长度（默认为 2048 位）
2. 点击"生成 RSA 密钥对"按钮
3. 等待密钥生成完成（可能需要几秒到几十秒）
4. 复制并妥善保存生成的公钥和私钥

### 密钥长度选择建议

- **1024 位**：仅用于测试目的，不推荐在生产环境中使用
- **2048 位**：当前推荐的最小密钥长度，适用于大多数场景
- **4096 位**：高安全性要求场景的选择，但生成时间较长

## 实现要点

1. 使用 Forge 库进行 RSA 密钥对生成
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 安全说明

1. 密钥生成过程完全在浏览器中进行，不会上传到任何服务器
2. 刷新页面或关闭浏览器将丢失生成的密钥
3. 请勿在不安全的网络环境下生成重要密钥
4. 私钥必须严格保密，一旦泄露将导致安全风险

## 注意事项

1. 密钥生成需要一定时间，特别是 4096 位密钥
2. 生成的密钥仅在当前浏览器会话中有效
3. 建议在安全的环境中生成和保存密钥
4. 对于生产环境使用，建议使用专业的密钥管理工具
