# JWT 解析与生成器

## 工具说明

JWT（JSON Web Token）解析与生成器是一个用于生成和解析 JWT Token 的在线工具。它支持多种 HMAC 算法（HS256、HS384、HS512）和 RSA 算法（RS256、RS384、RS512），可以在浏览器端直接解析 JWT，而对于 RSA 算法的签名功能，出于安全考虑暂不支持在浏览器中进行。

## 核心功能

1. **JWT 生成**：
   - 支持 HS256、HS384、HS512 算法
   - 可自定义 Header 和 Payload 内容
   - 实时显示生成的 JWT 及其组成部分

2. **JWT 解析**：
   - 解析 JWT Token 的 Header、Payload 和 Signature
   - 格式化显示解析结果
   - 验证 JWT 签名的有效性（仅支持 HMAC 算法）

## 使用说明

### 生成 JWT

1. 选择所需的算法（默认为 HS256）
2. 输入密钥（对于 HMAC 算法）或私钥（对于 RSA 算法，但浏览器端暂不支持）
3. 编辑 Header 和 Payload 的 JSON 内容
4. 点击"生成 JWT"按钮
5. 查看生成的 JWT 及其分段解析结果

### 解析 JWT

1. 在文本框中粘贴要解析的 JWT Token
2. 输入用于验证签名的密钥（对于 HMAC 算法）或公钥（对于 RSA 算法，但浏览器端暂不支持）
3. 点击"解析 JWT"按钮
4. 查看解析结果及签名验证状态

## 实现要点

1. 使用 Web Crypto API 实现 JWT 签名生成和验证
2. 集成 jwt-js-decode 库用于 JWT 解析
3. 采用 Bootstrap 5 构建响应式界面
4. 完全在客户端运行，保护用户数据隐私
5. 符合工具集统一 UI 风格和导航结构

## 安全说明

出于安全考虑，本工具暂不支持在浏览器中进行 RSA 算法的签名和验证操作。这是因为 RSA 操作涉及复杂的密钥管理和潜在的安全风险。建议在后端服务器环境中进行 RSA 签名操作。

## 可扩展方向

1. 支持更多算法（如 ES256、PS256 等）
2. 添加 JWT 过期时间检查
3. 增加 JWT 模板库功能
