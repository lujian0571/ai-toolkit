# Base64加解密工具

## 工具说明
这是一个基于浏览器的Base64编码和解码工具，用户可以在本地直接使用，无需连接网络或服务器。工具提供了编码、解码、内容交换和清空等功能。

## 核心功能点
1. Base64编码：将普通文本转换为Base64编码格式
2. Base64解码：将Base64编码还原为原始文本
3. 内容交换：快速交换输入和输出区域的内容
4. 清空功能：一键清除输入和输出区域的内容
5. 复制结果：将处理结果复制到剪贴板

## 使用说明
1. 在输入框中输入需要处理的文本
2. 点击"Base64 编码"按钮将文本转换为Base64格式
3. 点击"Base64 解码"按钮将Base64编码还原为原始文本
4. 使用"交换内容"按钮可以在输入和输出框之间交换内容
5. 使用"清空"按钮可以清除所有内容
6. 使用"复制结果"按钮可以将输出结果复制到剪贴板

## 设计思路
1. 采用单输入单输出的设计，简化用户操作流程
2. 集成编码和解码功能在一个界面中，提高使用效率
3. 添加实用的辅助功能如内容交换和清空按钮
4. 遵循Bootstrap设计规范，保证界面美观和响应式
5. 使用亮色主题，提供良好的用户体验
