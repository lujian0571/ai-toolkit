# 密码生成器

密码生成器是一个安全、灵活的工具，用于生成高强度的随机密码。该工具支持自定义密码长度和字符类型，帮助用户创建符合安全要求的密码。

## 工具规范

### 菜单要求
- 工具应归类于"生成工具"分类下
- 菜单项名称为"密码生成"
- 面包包屑导航应为：生成工具 / 密码生成

### 界面要求
- 使用卡片组件展示内容，每个卡片包含 card-header 区域
- 使用 Bootstrap 5 作为 UI 框架
- 使用公共样式和脚本文件

### 功能要求
1. 密码生成设置：
   - 密码长度调节（4-50位）
   - 字符类型选择（大写字母、小写字母、数字、符号）
   - 排除选项（相似字符、模糊字符）

2. 密码生成与显示：
   - 参数调整后应自动重新生成密码
   - 提供"生成密码"按钮用于手动重新生成
   - 同时生成9组密码并以3x3网格显示
   - 每个密码提供复制按钮
   - 提供保存所有密码按钮

3. 技术实现要求：
   - 使用 `window.crypto.getRandomValues()` 生成高质量随机数
   - 所有处理在浏览器端完成，不上传任何数据到服务器
   - 确保生成的密码包含所有选定的字符类型

## 功能特点

### 密码生成
- 自定义密码长度（4-50位）
- 多种字符类型选择：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 符号 (!@#$%^&*)
- 排除选项：
  - 排除相似字符 (0, O, l, I)
  - 排除模糊字符 ({}, []等)

### 密码强度评估
- 实时显示密码强度（弱/一般/较强/强）
- 可视化进度条显示强度百分比

### 密码操作
- 一键复制密码到剪贴板
- 保存密码到本地文件

## 技术实现

### 安全性
- 使用 `window.crypto.getRandomValues()` 生成高质量的随机数
- 确保生成的密码包含所有选定的字符类型

### 用户体验
- 实时生成密码（调整参数后立即生成新密码）
- 直观的界面设计和清晰的反馈提示
- 响应式布局，支持各种设备屏幕尺寸

## 使用说明

1. **设置密码参数**：
   - 调整滑块设置密码长度
   - 选择需要包含的字符类型
   - 根据需要选择排除选项

2. **生成密码**：
   - 参数调整后会自动重新生成密码
   - 也可点击"生成密码"按钮手动重新生成

3. **使用密码**：
   - 点击复制按钮将密码复制到剪贴板
   - 点击保存按钮将密码保存为文本文件

## 安全建议

1. 为不同账户生成不同的密码
2. 定期更换重要账户的密码
3. 使用较长且复杂的密码（建议12位以上）
4. 结合多种字符类型以提高密码强度
5. 避免在密码中使用个人信息（如生日、姓名等）

## 注意事项

1. 生成的密码只在浏览器本地生成，不会上传到任何服务器
2. 保存的密码文件仅存储在本地设备上
3. 请妥善保管生成的密码，一旦丢失无法找回
4. 建议配合密码管理器使用以获得更好的安全性
