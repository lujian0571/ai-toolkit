# 随机数生成器工具

## 工具说明

随机数生成器工具是一个用于生成各种类型随机数的在线工具。支持整数、浮点数和不重复整数等多种随机数类型，并允许用户自定义数值范围和生成数量。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **多种随机数类型**：
   - 整数随机数生成
   - 浮点数随机数生成
   - 不重复整数序列生成

2. **灵活配置**：
   - 自定义数值范围（最小值和最大值）
   - 设置生成数量（1-1000个）
   - 浮点数小数位数设置

3. **结果处理**：
   - 直观的结果展示界面
   - 一键复制功能
   - 清空结果功能

## 使用说明

### 基本使用

1. 设置随机数的最小值和最大值范围
2. 选择需要的数字类型（整数、浮点数或不重复整数）
3. 如果需要浮点数，可以设置小数位数
4. 设置要生成的随机数数量
5. 点击"生成随机数"按钮获取结果

### 功能说明

- **整数**：生成指定范围内的随机整数
- **浮点数**：生成指定范围内的随机浮点数，并可指定小数位数
- **不重复整数**：生成指定范围内的不重复随机整数序列

2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构
5. 提供多种随机数类型以满足不同需求

## 技术细节

### 生成原理

随机数生成基于以下原理：
1. 使用浏览器内置的 `Math.random()` 函数生成 0 到 1 之间的伪随机数
2. 通过数学运算将随机数映射到指定范围
3. 对于不重复整数，使用 Set 数据结构确保唯一性

### 算法实现

```javascript
// 整数生成
Math.floor(Math.random() * (max - min + 1)) + min

// 浮点数生成
Math.round((Math.random() * (max - min) + min) * multiplier) / multiplier

// 不重复整数生成
使用 Set 数据结构跟踪已生成的数字
```

## 注意事项

1. 本工具使用浏览器内置的随机数生成器，适用于一般用途
2. 对于高安全性要求的应用（如密码学），需要使用专门的加密安全随机数生成器
3. 生成大量不重复整数时，需确保范围足够大以避免性能问题
4. 工具完全在浏览器中运行，不依赖后端服务
