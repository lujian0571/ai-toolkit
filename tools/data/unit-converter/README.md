# 单位换算器工具

## 工具说明

单位换算器工具是一个用于在不同物理量单位之间进行转换的在线工具。支持长度、重量、温度、面积、体积和速度等多种常见单位的换算。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **多类别单位换算**：
   - 长度单位换算（米、千米、厘米、英寸、英尺等）
   - 重量单位换算（克、千克、吨、盎司、磅等）
   - 温度单位换算（摄氏度、华氏度、开尔文）
   - 面积单位换算（平方米、平方千米、公顷、英亩等）
   - 体积单位换算（升、毫升、立方米、加仑等）
   - 速度单位换算（米/秒、千米/小时、英里/小时等）

2. **便捷操作**：
   - 支持输入输出单位互换
   - 实时换算结果展示
   - 直观的分类选择

3. **结果显示**：
   - 自动格式化数字显示
   - 对极大或极小数值使用科学记数法

## 使用说明

### 基本使用

1. 选择要换算的单位类别
2. 输入要转换的数值
3. 选择输入单位和目标单位
4. 查看自动计算的换算结果

### 特殊功能

- **交换单位**：点击中间的交换按钮可以快速交换输入单位和输出单位
- **实时换算**：输入数值后会自动计算结果
- **分类切换**：可以通过单位类别下拉菜单切换不同的单位类型

## 实现要点

1. 使用原生 JavaScript 实现单位换算逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构
5. 特殊处理温度换算（非线性换算）

## 技术细节

### 换算原理

单位换算基于以下原理：
1. 标准单位换算：通过相对于国际标准单位的换算系数实现
2. 温度换算：使用专门的换算公式处理摄氏度、华氏度和开尔文之间的转换

### 数据结构

使用对象结构存储各类单位及其换算因子：
```javascript
{
  length: {
    name: "长度",
    units: {
      "m": { name: "米", factor: 1 },
      "km": { name: "千米", factor: 1000 },
      // ...
    }
  }
  // ...
}
```

## 注意事项

1. 输入数值必须是有效的数字
2. 温度换算是特殊情况，不使用简单的比例换算
3. 工具完全在浏览器中运行，不依赖后端服务
4. 对于极高精度的需求，可能需要专业工具
