# 网络和IP地址计算器

## 工具介绍

网络和IP地址计算器是一个用于计算IPv4网络相关信息的实用工具。它可以帮助网络管理员、开发人员和网络安全专业人员快速获取IP地址的各种网络属性，包括网络地址、广播地址、可用主机范围等重要信息。

## 核心功能

1. **IP地址信息计算**：
   - 输入任意有效的IPv4地址和子网掩码
   - 自动计算网络地址、广播地址
   - 显示可用主机范围和主机数量
   - 识别IP地址类别（A、B、C、D、E类）

2. **多种子网掩码格式支持**：
   - 支持CIDR表示法（如/24）
   - 支持点分十进制表示法（如255.255.255.0）
   - 提供常用的预设子网掩码选项

3. **详细网络信息展示**：
   - 以文本形式展示基本网络信息
   - 以表格形式展示详细的子网参数

4. **知识科普**：
   - 解释IP地址和子网掩码的基本概念
   - 提供常见子网掩码对照表
   - 介绍工具使用方法

## 使用说明

1. 在"IP 地址"输入框中输入有效的IPv4地址（如192.168.1.1）
2. 在"子网掩码"下拉菜单中选择预设值或选择"自定义"并输入自定义子网掩码
3. 点击"计算网络信息"按钮
4. 查看计算结果，包括网络地址、广播地址、可用主机范围等信息

## 实现要点

- 使用原生JavaScript实现所有网络计算逻辑，无需后端支持
- 采用Bootstrap 5构建响应式用户界面，适配各种设备
- 实现了完整的输入验证和错误处理机制
- 遵循项目规范，与其他工具保持一致的UI风格
