# 进制转换工具

## 工具说明

进制转换工具是一个用于在不同数制系统之间进行转换的在线工具。支持二进制、八进制、十进制和十六进制之间的相互转换。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **进制转换**：
   - 二进制 (Binary) 转换
   - 八进制 (Octal) 转换
   - 十进制 (Decimal) 转换
   - 十六进制 (Hexadecimal) 转换

2. **输入验证**：
   - 验证输入数值是否符合所选进制的规则
   - 提供清晰的错误提示

3. **结果显示**：
   - 同时显示转换到所有四种进制的结果
   - 使用等宽字体显示数值，便于阅读

## 使用说明

### 基本使用

1. 在输入框中输入要转换的数值
2. 选择输入数值的进制类型
3. 点击"执行转换"按钮
4. 在下方查看转换到各种进制的结果

### 支持的进制

- **二进制 (Binary)**：基数为2，仅使用0和1两个数字
- **八进制 (Octal)**：基数为8，使用0到7八个数字
- **十进制 (Decimal)**：基数为10，使用0到9十个数字
- **十六进制 (Hexadecimal)**：基数为16，使用0到9和A到F十六个符号

## 实现要点

1. 使用原生 JavaScript 实现进制转换逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 转换原理

进制转换基于以下原理：
1. 首先将输入数值转换为十进制中间值
2. 然后将十进制值转换为目标进制
3. 使用 JavaScript 内置的 parseInt() 和 toString(radix) 方法实现转换

### 输入验证

根据不同进制验证输入：
- 二进制：只允许包含 0 和 1
- 八进制：只允许包含 0-7
- 十进制：只允许包含 0-9
- 十六进制：只允许包含 0-9 和 A-F（不区分大小写）

## 注意事项

1. 输入数值必须符合所选进制的规则
2. 工具完全在浏览器中运行，不依赖后端服务
3. 对于超大数值，可能会有精度限制
