# 颜色值转换工具

## 工具说明

颜色值转换工具是一个用于在不同颜色表示格式之间进行转换的在线工具。支持 HEX、RGB 和 HSL 三种常见的颜色格式之间的相互转换。完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **颜色格式转换**：
   - HEX (#RRGGBB) 格式转换
   - RGB (rgb(r, g, b)) 格式转换
   - HSL (hsl(h, s%, l%)) 格式转换

2. **输入验证**：
   - 验证输入颜色值是否符合所选格式的规则
   - 提供清晰的错误提示

3. **结果显示**：
   - 同时显示转换到所有三种格式的结果
   - 使用等宽字体显示颜色值，便于阅读
   - 提供颜色预览功能

4. **智能检测**：
   - 自动检测输入颜色值的格式

## 使用说明

### 基本使用

1. 在输入框中输入要转换的颜色值
2. 选择输入颜色值的格式或使用自动检测功能
3. 点击"执行转换"按钮
4. 在下方查看转换到各种格式的结果

### 支持的颜色格式

- **HEX (十六进制)**：以 # 开头，后跟六个十六进制字符 (0-9, A-F)，分别表示红、绿、蓝三个通道的值。例如 #FF0000 表示红色。
- **RGB (红绿蓝)**：通过指定红(Red)、绿(Green)、蓝(Blue)三个通道的值来表示颜色，每个通道的取值范围是 0-255。例如 rgb(255, 0, 0) 表示红色。
- **HSL (色相、饱和度、亮度)**：通过色相(Hue)、饱和度(Saturation)、亮度(Lightness)来表示颜色。色相取值范围为 0-360 度，饱和度和亮度取值范围为 0%-100%。例如 hsl(0, 100%, 50%) 表示红色。

## 实现要点

2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构
5. 提供实时颜色预览功能

## 技术细节

### 转换原理

颜色格式转换基于以下原理：
1. 不同颜色格式间的转换通过数学公式实现
2. RGB 和 HEX 格式之间可以直接转换
3. RGB 和 HSL 格式之间通过三角函数计算进行转换

### 输入验证

根据不同格式验证输入：
- HEX 格式：必须是以 # 开头的 6 位十六进制数
- RGB 格式：必须是 rgb(r, g, b) 形式，且每个值在 0-255 范围内
- HSL 格式：必须是 hsl(h, s%, l%) 形式，且 h 在 0-360 范围内，s 和 l 在 0-100% 范围内

## 注意事项

1. 输入颜色值必须符合所选格式的规则
2. 工具完全在浏览器中运行，不依赖后端服务
3. 对于特殊的颜色值（如透明度），可能需要专业工具
