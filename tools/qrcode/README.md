# 二维码工具

本工具包含两个功能模块：二维码生成器和二维码识别器，可以帮助用户快速生成和识别二维码。

## 功能介绍

### 二维码生成器

二维码生成器支持以下功能：
- 自定义二维码内容（文本、网址等）
- 调整二维码尺寸（100-500像素）
- 自定义前景色和背景色
- 一键下载生成的二维码图片

### 二维码识别器

二维码识别器支持以下功能：
- 文件上传识别（支持常见的图片格式）
- 拖拽图片识别
- 粘贴图片识别（从剪贴板）
- 一键复制识别结果

## 技术实现

### 二维码生成器

使用 [qrcode](https://github.com/soldair/node-qrcode) 库实现二维码生成，该库支持丰富的自定义选项：
- 可调节尺寸和边距
- 可自定义前景色和背景色
- 输出为 Canvas 或 DataURL 格式

### 二维码识别器

使用 [jsQR](https://github.com/cozmo/jsQR) 库实现二维码识别，该库特点：
- 纯 JavaScript 实现，无需后端支持
- 支持多种图片格式
- 识别速度快，准确率高

## 使用说明

### 二维码生成器

1. 在内容输入框中输入要生成二维码的内容（如网址、文本等）
2. 调整尺寸滑块选择合适的二维码大小
3. 选择前景色和背景色（可选）
4. 点击"生成二维码"按钮
5. 预览生成的二维码，点击"下载二维码"保存到本地

### 二维码识别器

1. 选择以下任一方式上传二维码图片：
   - 点击"选择文件"按钮上传本地图片
   - 将二维码图片拖拽到指定区域
   - 点击粘贴区域并按下 Ctrl+V（Mac 用户使用 Cmd+V）粘贴图片
2. 系统会自动识别二维码内容
3. 如果识别成功，会在右侧显示识别结果
4. 点击复制按钮可将结果复制到剪贴板

## 注意事项

1. 二维码识别的准确性取决于图片质量和二维码清晰度
2. 建议使用对比度明显的颜色组合以确保二维码可扫描
3. 生成的二维码图片为 PNG 格式，保留透明背景（如果设置了透明背景色）
4. 所有处理都在浏览器端完成，不会上传任何图片到服务器，保证用户隐私安全
