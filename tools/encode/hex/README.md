# Hex 编码/解码工具

## 工具说明

Hex 编码/解码工具是一个用于在字符串和十六进制表示之间进行转换的在线工具。该工具完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **Hex 编码**：
   - 将字符串转换为其十六进制表示形式
   - 支持所有 Unicode 字符

2. **Hex 解码**：
   - 将十六进制字符串转换回原始字符串
   - 支持大小写字母的十六进制字符

3. **辅助功能**：
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### Hex 编码

1. 在输入框中输入需要编码的文本
2. 点击"Hex 编码"按钮
3. 在输出框中查看十六进制结果

### Hex 解码

1. 在输入框中输入十六进制字符串
2. 点击"Hex 解码"按钮
3. 在输出框中查看解码后的文本

## 实现要点

1. 使用原生 JavaScript 实现编码和解码逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 编码过程

1. 遍历输入字符串中的每个字符
2. 获取字符的 Unicode 码点
3. 将码点转换为两位十六进制数
4. 连接所有十六进制数形成最终结果

### 解码过程

1. 验证输入是否为有效的十六进制字符串
2. 验证字符串长度是否为偶数
3. 每两位十六进制数为一组进行处理
4. 将每组十六进制数转换为对应的字符
5. 连接所有字符形成最终结果

## 注意事项

1. 解码时输入必须是有效的十六进制字符串（0-9, A-F, a-f）
2. 解码时字符串长度必须为偶数（每两个字符表示一个字节）
3. 无效的十六进制字符会导致解码失败
4. 工具完全在浏览器中运行，不依赖后端服务
