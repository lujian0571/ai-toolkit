关于 Unicode 编码
什么是 Unicode

Unicode 是计算机科学领域里的一项业界标准，它对世界上大部分的文字系统进行了整理、编码， 使得电脑可以用更为简单的方式来呈现和处理文字。

Unicode 编码方式
  • \uXXXX 格式：最常见的 Unicode 转义序列，使用 4 位十六进制数表示字符
  • &#xXXXX; 格式：HTML/XML 中的十六进制实体引用
  • &#XXXX; 格式：HTML/XML 中的十进制实体引用
常见用途
  • 在程序中表示非 ASCII 字符
  • 在网页中显示特殊字符
  • 在 JSON 等数据格式中处理国际化字符
  • 在正则表达式中匹配特定字符
如何使用本工具
  1. 编码：在输入框中输入原文，选择编码格式，点击"Unicode 编码"按钮
  2. 解码：在输入框中输入 Unicode 编码字符串，选择对应格式，点击"Unicode 解码"按钮
  3. 格式选择：可以选择不同的编码格式进行转换
  4. 交换内容：点击"交换内容"按钮可以将输入框和输出框的内容互换，方便连续操作
  5. 清空：点击"清空"按钮可以同时清空所有输入框和输出框的内容
  6. 复制结果：点击"复制结果"按钮可以将输出框中的内容复制到剪贴板
注意事项
  • 解码时需要选择与编码时相同的格式
  • 某些格式可能需要特殊的转义处理
  • 超出基本多文种平面的字符可能需要代理对表示
