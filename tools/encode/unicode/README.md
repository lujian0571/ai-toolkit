# Unicode 编码/解码工具

## 工具说明

Unicode 编码/解码工具是一个用于在字符串和 Unicode 编码表示之间进行转换的在线工具。该工具支持多种 Unicode 编码格式，完全在浏览器中运行，不会将用户数据发送到任何服务器，确保用户隐私安全。

## 核心功能

1. **Unicode 编码**：
   - 支持 \uXXXX 格式编码
   - 支持 HTML 十六进制实体 &#xXXXX; 格式编码
   - 支持 HTML 十进制实体 &#XXXX; 格式编码

2. **Unicode 解码**：
   - 支持对应格式的解码操作
   - 支持所有 Unicode 字符

3. **辅助功能**：
   - 多种编码格式选择
   - 内容交换：方便连续操作
   - 清空功能：快速清除所有输入输出
   - 复制结果：一键复制到剪贴板

## 使用说明

### Unicode 编码

1. 在输入框中输入需要编码的文本
2. 选择所需的编码格式
3. 点击"Unicode 编码"按钮
4. 在输出框中查看编码结果

### Unicode 解码

1. 在输入框中输入 Unicode 编码字符串
2. 选择对应的编码格式
3. 点击"Unicode 解码"按钮
4. 在输出框中查看解码后的文本

## 实现要点

1. 使用原生 JavaScript 实现编码和解码逻辑
2. 采用 Bootstrap 5 构建响应式界面
3. 完全在客户端运行，保护用户数据隐私
4. 符合工具集统一 UI 风格和导航结构

## 技术细节

### 支持的编码格式

1. **\uXXXX 格式**：
   - 最常见的 Unicode 转义序列
   - 使用 4 位十六进制数表示字符
   - 例如：中文"中"编码为\u4E2D

2. **&#xXXXX; 格式**：
   - HTML/XML 中的十六进制实体引用
   - 以 &#x 开头，以 ; 结尾
   - 例如：中文"中"编码为&#x4E2D;

3. **&#XXXX; 格式**：
   - HTML/XML 中的十进制实体引用
   - 以 &# 开头，以 ; 结尾
   - 例如：中文"中"编码为&#20013;

## 注意事项

1. 编码和解码时需要选择对应的格式
2. 工具完全在浏览器中运行，不依赖后端服务
3. 对于超出基本多文种平面的字符可能需要特殊处理
4. 输入格式错误可能导致解码失败
